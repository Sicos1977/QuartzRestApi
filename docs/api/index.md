# API Reference

Browse the full public API of **QuartzRestApi** in the navigation on the left.

The two main entry points are:

| Type | Description |
|---|---|
| [`SchedulerHost`](xref:QuartzRestApi.SchedulerHost) | Bootstraps the self-hosted REST API server around an existing `IScheduler`. |
| [`SchedulerConnector`](xref:QuartzRestApi.SchedulerConnector) | Typed HTTP client that covers every endpoint exposed by `SchedulerHost`. |
